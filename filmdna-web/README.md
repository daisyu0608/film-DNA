# filmDNA · 电影审美图谱

输入你最爱的 5–10 部电影，AI 算出你的导演谱系坐标 / 16 型电影人格 / 专属片单。

## 算法基础

整合九大经典编剧理论体系，以鲍德威尔《电影艺术：形式与风格》四象限为骨架：

- **古典叙事** Classical Narrative
- **艺术电影** Art Cinema
- **参数化** Parametric
- **后古典** Post-Classical

## 输出维度

- 4 维核心向量（视觉 / 镜头 / 叙事 / 类型）
- 16 子维度详细诊断
- 6 个传播性诊断标签
- 鲍德威尔象限定位
- Top 3 最像的导演
- 10 部专属推荐片单
- 1080×1920 可分享卡片

## 数据规模

- 1809 部精标电影库
- 515 位世界电影导演坐标
- 16 型电影人格

## 技术栈

纯前端，无后端依赖。HTML + CSS + 原生 JavaScript。
